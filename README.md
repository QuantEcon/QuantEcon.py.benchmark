# QuantEcon.py.benchmark

Contains benchmarks for the QuantEcon.py repository using the asv project

## Commands

To run the timing tests

```bash
asv run
```

To generate the html

```bash
asv publish
```

To preview the html through a python http server

```bash
asv preview
```
